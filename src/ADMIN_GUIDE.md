## 📋 Managing Reservations

### View Reservations

**Click "Rezervace" tab**

You'll see:
- 📊 **Statistics:** Total, Pending, Confirmed, Cancelled
- 🎯 **Filters:** All / Pending / Confirmed / Cancelled
- 📝 **List:** All reservations with details

### Reservation Card Shows:

- Status badge (colored)
- Guest name
- Email & phone
- Date & time
- Number of guests
- Optional message

### Actions You Can Take:

#### ✅ Confirm Reservation
Click green "Potvrdit" button
→ Status changes to "Potvrzeno"

#### ❌ Cancel Reservation
Click red "Zrušit" button
→ Status changes to "Zrušeno"

#### 🗑️ Delete Reservation
Click "Smazat" button
→ Confirmation dialog appears
→ Click OK to permanently delete

#### 🔄 Refresh
Click "Obnovit" button
→ Reloads latest reservations from server

## 📅 Weekly Menu

**Click "Týdenní menu" tab**

### Add Daily Specials:

1. Select week (Monday date)
2. For each day:
   - Click "+" button
   - Enter dish name
   - Enter price
   - Click "Přidat"
3. Click "Uložit menu"

### Edit Items:
- Click pencil icon
- Change name/price
- Click "Uložit"

### Delete Items:
- Click trash icon
- Item removed

---

## 📖 Main Menu

**Click "Hlavní menu" tab**

### View Menu:
- See all categories
- See all dishes with prices

### Edit Dish:
1. Click pencil icon
2. Edit name (Czech/English/Italian)
3. Edit description (Czech/English/Italian)
4. Edit price
5. Click "Uložit"

### Add New Dish:
1. Select category
2. Click "Přidat položku"
3. Fill in all fields
4. Click "Uložit"

### Delete Dish:
- Click trash icon
- Confirm deletion

## ❓ Troubleshooting

### Can't Log In:
- Check password is correct: `menicka2026`
- Try clearing browser cache
- Try different browser

### Reservations Not Showing:
- Click "Obnovit" (Refresh)
- Check internet connection
- Try logging out and back in

### Can't Confirm/Delete:
- Check internet connection
- Refresh page
- Try again

### Email Not Received:
- Check spam folder
- Email may be delayed
- Reservation still saved in system
