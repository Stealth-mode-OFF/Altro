# 🔑 Jak získat Resend API Key

## 📋 Rychlý návod (2 minuty)

### Krok 1: Registrace
1. Jděte na **https://resend.com**
2. Klikněte **"Sign Up"**
3. Zaregistrujte se pomocí GitHub nebo emailu

### Krok 2: Vytvoření API klíče
1. Po přihlášení klikněte na **"API Keys"** v levém menu
2. Klikněte **"Create API Key"**
3. Pojmenujte klíč: **"Altro Da Tony Production"**
4. Vyberte **"Full Access"** (nebo "Sending Access")
5. Klikněte **"Create"**
6. **Zkopírujte klíč** - začíná `re_...`

### Krok 3: Přidání do Supabase
1. Otevřete Figma Make (tento projekt)
2. Najděte modal pro **RESEND_API_KEY**
3. Vložte zkopírovaný klíč
4. Klikněte **"Save"**
