# ⚡ QUICKSTART - Altro Da Tony

## 🚀 Spusťte web za 5 minut!

### Krok 1: Push na GitHub (1 min)
```bash
# V Figma Make klikněte na modré "GitHub" tlačítko
# ✅ Hotovo!
```

### Krok 2: Deploy na Vercel (2 min)
1. Jděte na **https://vercel.com**
2. Klikněte **"New Project"**
3. Import `Stealth-mode-OFF/altrodatony`
4. Klikněte **"Deploy"**
5. ✅ Web je LIVE!

### Krok 3: Test (2 min)
- Otevřete web na `your-project.vercel.app`
- Vytvořte testovací rezervaci
- Přihlaste se do admin (`/admin`, heslo: `altrodatony2024`)
- Zkontrolujte rezervaci v admin panelu
